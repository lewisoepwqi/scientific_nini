## ADDED Requirements

### Requirement: 图表产物必须输出标准导出 metadata

系统 SHALL 为每个正式图表产物输出标准导出 metadata，至少包含图表资源标识、样式模板、分辨率、导出格式与生成时间。

#### Scenario: 图表导出时附带 metadata
- **WHEN** 图表完成正式导出
- **THEN** 系统为该图表生成标准导出 metadata
- **AND** metadata 可被项目产物清单直接消费

#### Scenario: 图表进入项目产物清单
- **WHEN** 图表导出结果登记为项目产物
- **THEN** 产物记录包含图表类型、导出格式与样式模板信息

### Requirement: 图表导出必须提供稳定格式集合

系统 SHALL 为图表导出提供稳定格式集合，并向产物层声明哪些格式已成功生成或发生降级。

#### Scenario: 图表导出多个格式
- **WHEN** 图表导出作业执行成功
- **THEN** 系统返回该次导出的成功格式列表
- **AND** 这些格式可在项目产物清单中查询

#### Scenario: 图表格式降级被记录
- **WHEN** 某个目标格式导出失败但其他格式成功
- **THEN** 系统记录降级结果
- **AND** 产物清单中可见成功与失败的格式信息
