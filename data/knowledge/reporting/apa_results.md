# APA 格式统计结果报告模板

## 基本规则

- 统计符号用斜体：*t*, *F*, *p*, *r*, *M*, *SD*, *n*
- p 值小于 .001 时报告为 *p* < .001，不写 *p* = .000
- 效应量必须报告
- 小数点前的零：能超过 1 的指标保留（如 *M* = 0.35），不能超过 1 的省略（如 *p* = .032, *r* = .45）

## 报告模板

### t 检验
独立样本：*t*(df) = X.XX, *p* = .XXX, *d* = X.XX
配对样本：*t*(df) = X.XX, *p* = .XXX, *d* = X.XX

示例：实验组（*M* = 4.52, *SD* = 1.23）显著高于对照组（*M* = 3.18, *SD* = 1.45），*t*(58) = 3.76, *p* < .001, *d* = 0.97。

### ANOVA
*F*(df_between, df_within) = X.XX, *p* = .XXX, η² = .XX

示例：三组间差异显著，*F*(2, 87) = 6.42, *p* = .002, η² = .13。Tukey HSD 事后检验表明，A 组显著高于 C 组（*p* = .001）。

### 相关
*r*(df) = .XX, *p* = .XXX

示例：焦虑得分与睡眠质量呈显著负相关，*r*(98) = -.42, *p* < .001。

### 回归
*F*(df_regression, df_residual) = X.XX, *p* = .XXX, *R*² = .XX

示例：回归模型显著，*F*(3, 96) = 12.34, *p* < .001, *R*² = .28，三个预测变量共解释了 28% 的方差。

### 卡方检验
χ²(df, *N* = XX) = X.XX, *p* = .XXX

示例：性别与专业选择之间存在显著关联，χ²(3, *N* = 200) = 15.67, *p* = .001, Cramér's *V* = .28。
