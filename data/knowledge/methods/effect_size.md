# 效应量计算与解读指南

## 为什么要报告效应量

- p 值只说明"有没有差异"，不说明"差异有多大"
- 大样本下微小差异也能"统计显著"
- 期刊（APA 第 7 版）要求报告效应量

## 常用效应量指标

### 两组比较
| 指标 | 计算 | 适用方法 |
|------|------|---------|
| Cohen's d | (M₁ - M₂) / SD_pooled | t 检验 |
| Hedges' g | 修正小样本偏差的 d | 小样本 t 检验 |
| r (effect size r) | 点二列相关 | Mann-Whitney U |

### 多组比较
| 指标 | 计算 | 适用方法 |
|------|------|---------|
| η² (eta squared) | SS_between / SS_total | ANOVA |
| partial η² | SS_effect / (SS_effect + SS_error) | 多因素 ANOVA |
| ω² (omega squared) | 偏差修正的 η² | ANOVA（更推荐） |

### 相关与回归
| 指标 | 说明 |
|------|------|
| r | Pearson 相关系数本身就是效应量 |
| R² | 回归解释的方差比例 |
| f² | Cohen's f²，用于回归 |

## 解读标准（Cohen, 1988）

| 效应量 | 小 | 中 | 大 |
|--------|-----|-----|-----|
| d | 0.2 | 0.5 | 0.8 |
| r | 0.1 | 0.3 | 0.5 |
| η² | 0.01 | 0.06 | 0.14 |
| f² | 0.02 | 0.15 | 0.35 |

注意：Cohen 的标准仅为参考。效应量的实际意义应结合具体研究领域判断。
