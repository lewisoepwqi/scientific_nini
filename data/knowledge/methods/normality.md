# 正态性检查与参数/非参数选择指南

## 正态性检验方法

| 方法 | 适用场景 | 说明 |
|------|---------|------|
| Shapiro-Wilk | n < 50 | 最常用，检验力最强 |
| Kolmogorov-Smirnov | n >= 50 | 大样本时使用 |
| D'Agostino-Pearson | n >= 20 | 综合偏度和峰度 |
| Q-Q 图 | 任何样本量 | 视觉判断，应与统计检验配合 |

## 正态性不满足时的策略

### 策略一：数据变换
- **右偏（正偏态）**：对数变换 log(x)、平方根变换 √x
- **左偏（负偏态）**：反射后对数变换
- **变换后重新检验正态性**

### 策略二：使用非参数替代

| 参数方法 | 非参数替代 |
|---------|-----------|
| 独立样本 t 检验 | Mann-Whitney U |
| 配对 t 检验 | Wilcoxon 符号秩 |
| 单因素 ANOVA | Kruskal-Wallis |
| Pearson 相关 | Spearman 相关 |

### 策略三：依赖中心极限定理
- 当 **n >= 30** 时，均值的抽样分布趋近正态
- 此时 t 检验和 ANOVA 对非正态性具有较好的稳健性
- 但极端偏态或存在离群值时仍需谨慎

## 判断流程

1. 画直方图和 Q-Q 图 → 视觉初步判断
2. 运行 Shapiro-Wilk 检验
3. 若 p > 0.05 → 可使用参数方法
4. 若 p <= 0.05 → 考虑样本量：
   - n >= 30 且偏度不严重 → 参数方法仍可接受
   - n < 30 或严重偏态 → 使用非参数方法或数据变换
