# 相关与回归方法选择指南

## 相关分析

| 条件 | 推荐方法 |
|------|---------|
| 连续变量 + 双变量正态 | Pearson 相关系数 |
| 有序变量 / 非正态 | Spearman 秩相关 |
| 小样本 / 有序分类 | Kendall's tau |
| 两个二分变量 | Phi 系数 |
| 一个二分 + 一个连续 | 点二列相关 |

## 回归分析

| 场景 | 推荐方法 |
|------|---------|
| 一个连续因变量 | 线性回归 |
| 二分类因变量 | Logistic 回归 |
| 计数因变量 | Poisson 回归 |
| 多个自变量 | 多元线性回归 |
| 需要变量筛选 | 逐步回归 / LASSO |

## 回归诊断

在解读回归结果前，必须检查：
1. **线性关系**：残差 vs 拟合值散点图
2. **正态性**：残差的 Q-Q 图
3. **同方差性**：残差的方差是否恒定
4. **多重共线性**：VIF < 10（理想 < 5）
5. **独立性**：Durbin-Watson 检验（时间序列数据）

## 相关系数解读

| |r| 范围 | 强度 |
|----------|------|
| 0.00 - 0.10 | 可忽略 |
| 0.10 - 0.30 | 弱相关 |
| 0.30 - 0.50 | 中等相关 |
| 0.50 - 0.70 | 强相关 |
| 0.70 - 1.00 | 很强相关 |

注意：相关不等于因果。高相关可能由混淆变量导致。
